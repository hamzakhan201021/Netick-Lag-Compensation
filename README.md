# Netick Lag Compensation

FREE Lag compensation system for Netick.

---

## Installation

### NOTE
To install this package you must have [Netick](https://github.com/NetickNetworking/NetickForUnity) installed.

### 1. Local Package, this method allows you to modify the package easily or even contribute
1. Clone this repository using [github desktop](https://desktop.github.com/download/) or any other git tool,
2. Open the Package Manager and click add,
3. Choose "install package from disk" and select the package.json file inside the cloned repository,
4. The package will be installed as a local package ready to be used.

### 2. Git URL, this method is easier and recommended if you want to use the package out of the box
1. Open the Package Manager and click add,
2. Choose "install package from git URL"
3. Enter the URL: https://github.com/hamzakhan201021/Netick-Lag-Compensation.git
4. Click install.
